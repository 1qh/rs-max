# rs-max

Maximum strictness Rust project, powered by [lintmax](https://github.com/1qh/lintmax-rs).

## Setup

```sh
cargo install cargo-lintmax
cargo lintmax init
```

## Commands

```sh
cargo lintmax          # check everything
cargo lintmax fix      # fix everything
cargo lintmax ci       # clean + update + check
cargo lintmax watch    # dev loop
cargo lintmax cov      # coverage
```
